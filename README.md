# GlobalLocalChat

Плагин для PaperMC, который добавляет локальный и глобальный чат на сервере Minecraft.

## Описание

GlobalLocalChat - это плагин для Minecraft серверов на базе PaperMC, который разделяет чат на локальный и глобальный режимы. Игроки могут общаться локально в радиусе или глобально по всему серверу.

## Особенности

- **Локальный чат**: Сообщения видны только игрокам в заданном радиусе
- **Глобальный чат**: Сообщения видны всем игрокам на сервере
- **Настраиваемые префиксы**: Цветные префиксы для разных типов сообщений
- **Кастомные сообщения входа/выхода**: Стилизованные уведомления о подключении/отключении игроков
- **Простая настройка**: Легко настраивается через config.yml

## Установка

1. Скачайте JAR файл плагина
2. Поместите файл `GlobalLocalChat-1.0.jar` в папку `plugins` вашего сервера
3. Перезапустите сервер или используйте команду `/reload`

## Конфигурация

После первого запуска плагина будет создан файл `config.yml` в папке `plugins/GlobalLocalChat/`.

```yaml
# Радиус локального чата
chat:
  local-radius: 150

# Цвета префиксов в формате HEX (#RRGGBB)
prefix-colors:
  global: '#00AAFF'    # [г]
  local: '#00FF00'     # [л]
  join: '#00FF00'      # [+]
  quit: '#FF5555'      # [-]
```

### Настройки

- `chat.local-radius`: Радиус локального чата в блоках
- `prefix-colors.global`: Цвет префикса глобального чата `[г]`
- `prefix-colors.local`: Цвет префикса локального чата `[л]`
- `prefix-colors.join`: Цвет префикса сообщений входа `[+]`
- `prefix-colors.quit`: Цвет префикса сообщений выхода `[-]`

## Использование

### Локальный чат
Просто пишите сообщения в чат - они будут видны только игрокам в радиусе 150 блоков (или настроенном значении).

### Глобальный чат
Начните сообщение с символа `!` для отправки всем игрокам на сервере.

```
Пример локального сообщения: Привет всем поблизости!
Пример глобального сообщения: !Всем привет на сервере!
```

## Сборка

Для сборки плагина вам потребуется Maven.

```bash
mvn clean package
```

Готовый JAR файл будет находиться в папке `target/`.

## Требования

- **Java**: 17 или выше
- **Сервер**: PaperMC 1.19+
